# 🚀gadgetfix — Gadget Repair & Accessories Platform

It is a modern, responsive web application for booking mobile/laptop repairs and purchasing accessories. Built with a clean, premium UI inspired by real startup products.

---

## ✨ Features

* 🔧 **Repair Booking System**
  Users can book repair services with device details and schedule.

* 🛒 **Accessories Shop**
  Browse and add products like chargers, headphones, and covers.

* 🎨 **Premium UI/UX**
  Clean, Apple-style design with smooth animations using Framer Motion.

* 📊 **Dynamic Stats Section**
  Animated counters showcasing trust and performance.

* 📱 **Fully Responsive**
  Optimized for mobile, tablet, and desktop devices.

* ⚡ **Fast Performance**
  Built using Vite + React for lightning-fast load times.

---

## 🛠️ Tech Stack

* **Frontend:** React, Vite
* **Styling:** Tailwind CSS
* **Animations:** Framer Motion
* **Icons:** Lucide React

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/fixify-plus.git
cd fixify-plus
```

### 2️⃣ Install dependencies

```bash
npm install
```

### 3️⃣ Run the development server

```bash
npm run dev
```

App will run on:

```
http://localhost:5173
```
